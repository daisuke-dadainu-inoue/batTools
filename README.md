# BatTools
Tools useful for business created in bat file
batファイルで作成されたビジネスに役立つツール

01_SQLServerStart --- SQLServer起動batファイル

02_CertificateCreation --- 証明書作成batファイル

03_DBBackUpForSQLServer --- DBバックアップbatファイル(SQLServer用, 1週間ローテート)

04_RegitJob --- タスクスケジューラー登録batファイル

05_ChangeAuthority --- 権限変更batファイル

06_FileBackUpFromLocalToServer --- ファイルバックアップbatファイル(ローカルからファイルサーバーへ, 1週間ローテート)
